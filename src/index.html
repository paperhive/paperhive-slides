The problem

You read a paper and

  • you don't understand what's going on
  • you think you found a mistake
  • there's a mistake and you didn't find it (but someone else did)
  • you know a better paper about the topic
  • you discovered an improvement and would like to contribute it to the original paper

Enter PaperHive

What can PaperHive do for you?

  • Review and improve
    • share your opinion on-the-fly
    • add helpful formulas, figures, data, code, ...
  • Discuss and learn
    • ask and answer questions
    • ignite fruitful conversations
  • Annotate and discover
    • attach your comments where they belong
    • find relevant comments immediately

What does it look like?

A network of papers and researchers

  • Comments may link to other papers, code, data, ...
  • Forward references, finally!
  • Structured discussions instead of unassociated comments

API + license

  • everyone can build tools on top of the PaperHive infrastructure with the PaperHive API
  • API documented on docs.paperhive.apiary.io
  • Public comments are licensed under Creative Commons BY license (CC-BY-4.0)

The future

Notifications for authors Allow discussion of paywalled papers Peer review process Linked data Tags Private comments (just for you or for a group) Use comments for alternative research metrics
